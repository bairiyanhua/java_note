JS:javascript
	运行在客户端上，基于对象和事件，有一定安全性的脚本语言
	
分类：
	行内脚本：直接在标签上写的js
		<p onclick="alert('我被点击了')">请给我一个口碑</p>
	内部脚本：
		用<script>标签包着，可以放在页面任何地方，一般放在<head>标签里
			<script type="text/javascript">
				document.write("<h1>请给我一个口碑</h1>");
			</script>
	外部脚本：
		新建一个以.js结尾的文件，然后从页面中用<script>标签的src属性引入
		注意：一旦引入，标签内部就不能写任何js脚本
			<script type="text/javascript" src="js/index.js"></script>

语法：
	1.声明变量统一用var来声明 
	2.数据类型
		undefined
		number
		string
		boolean
		object
		注意：使用typeof()可以判断变量的类型：var aa = 250; alert(typeof(aa));
	3.数组：
		声明：
			1).使用new Array声明并分配空间，通过下标赋值：var aa = new Array(5);
			2).使用new Array直接赋值：var aa = new Array("张三","李四","王五");
			3).使用[]直接赋值：var aa = ["张三","李四","王五"];
		属性和方法：
			属性：length: 长度
			方法：
				join(val): 把数组里的所有元素通过指定的val连接起来，变成字符串
				sort(): 把数字数组从小到大排序
				push(val)：在数组的后面添加指定的val值，变成一个新长度的数组
	4. 运算符
		+ - * / % ++ -- += -= > < >= <= != == === && || !
		==和===区别：
			都是比较两个值是否相等，但===还会去比较类型是否相等
		在JS中，数值和字符串比较是否相等，都用==
	5. 逻辑控制
		if(){}
		if(){}else{}
		if(){}else if(){}else{}
		switch(){case xx: break;}
		
		while(){}
		do{}while()
		for(){}
		
		增强for:
			for(var c in cc){
				alert(cc[c]);
			}
			注意：在js中，使用in，js中得到的不是具体的值，而是对就的下标
		中断循环：break  continue
	6.注释：
		单行注释： // 
		多行注释： /* ...  */
	7.弹框
		1)  普通弹框： alert()
			有一个参数，参数为想要提示的语句
			有一个“确定”按钮，点击关闭弹框
		2)  确认提示框： confirm()
			有一个参数，参数为想要提示的语句
			有“确定”和“取消”两个按钮，点确定，返回true，点取消，返回false
		3)  带输入框的弹框： prompt()
			有一个或两个参数，一个参数为想要提示的语句，两个参数后一个为输入框中的默认值
			有“确定”和“取消”两个按钮，点确定，将输入的值返回到js中，点取消，返回null
	
函数：
	系统函数：
		parseInt(): 把指定值转成整数
		parseFloat()： 把指定值转成小数
		isNaN(): 判断指定值是不是非数字，如果不是数字，返回true
	自定义函数：	
		function 函数名(){
		
		}
		function 函数名(参数名,...){
		
		}
	一般是配合着js事件，对函数进行调用：
		事件：
			onclick: 鼠标点击事件
			onfocus: 聚焦事件
			onblur: 失焦事件
			onkeydown: 键盘按下事件
			onchange: 内容改变事件
			onmouseenter: 鼠标移入事件
			onmouseleave： 鼠标移出事件
			
			onmouseover: 鼠标移入事件
			onmouseout: 鼠标移出事件
			onmousemove： 鼠标移动事件
