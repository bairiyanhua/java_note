java:
	java SE: java的基础，包含一系列的接口
	java EE: 提供了一系列的企业级的解决方案
	java ME: 移动端

java的执行过程：
	1. 我们自己写.java结尾的类
	2. 执行时，编译器会把.java文件编译成.class文件(半成品)
	3. 由虚拟机(JVM)把.class半成品编译成成品，在各个平台运行

注意：
	1. 类名和文件名一定要一样
	2. main方法的四要素顺序大小写一个都没能错
	3. 引号(")和结尾分号(;)必须是英文半角状态下的
	4. {}大括号都是成对出现的，前期写的所有代码，都在main方法的{}里面
